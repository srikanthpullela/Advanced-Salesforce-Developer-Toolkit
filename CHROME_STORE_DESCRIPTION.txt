Advanced Salesforce Developer Toolkit — The Ultimate Productivity Extension for Salesforce Developers & Admins

🔍 GLOBAL SEARCH (Cmd/Ctrl+Shift+P)
Instantly search across all Salesforce metadata — custom objects, fields, Apex classes, Apex triggers, Visualforce pages, Lightning components, flows, permission sets, profiles, validation rules, workflow rules, and more. Works like VS Code command palette right inside Salesforce.

🔎 RECORD INSPECTOR
Inspect any Salesforce record in real time. View all field values, field API names, field types, record type, owner, created/modified dates, and system fields. Copy field API names with one click. Perfect for debugging and troubleshooting data issues in Sales Cloud, Service Cloud, and any Salesforce org.

📝 SOQL QUERY EDITOR
Write and execute SOQL queries directly from any Salesforce page. Features syntax highlighting, auto-complete for object and field names, query history, and export results to CSV. No more switching to Developer Console or Workbench for quick queries.

🚀 QUICK NAVIGATOR
Navigate to any Salesforce Setup page instantly. Jump to object manager, custom objects, custom fields, page layouts, record types, Apex classes, Apex triggers, Lightning components, flows, process builder, permission sets, profiles, roles, users, and all admin setup pages with keyboard shortcuts.

🐛 APEX DEBUG LOG VIEWER
View and monitor Apex debug logs in real time without leaving your current page. Filter by log level, user, operation, and time range. Search within logs. No need to open Developer Console separately.

⚡ EXECUTE ANONYMOUS
Run Apex code snippets (execute anonymous) directly from any Salesforce page. Test Apex code, run data fixes, execute batch jobs, and debug issues without switching to Developer Console.

📊 DATA BUILDER
Create and manage test data directly in Salesforce. Build records for any standard or custom object. Ideal for Salesforce QA testing, sandbox data setup, and development.

🎯 CONTEXT MENU INTEGRATION
Right-click on any Salesforce page to quickly search selected text, inspect the current record, or open SOQL query tool.
